About me

I’m Kat Wlodarczyk. Originally from Poland but living permanently in UK since 2013. Web designer and developer.
I wrote my first lines of code when I was about 11 and needed a theme for a blog. It was a silly Paint-made theme with code written in NotePad, but I will never forget that feeling of pride and joy when I finished and it worked!
I still get this feeling everytime I design and develop.

Multi-tasker and work-a-holic. Photography lover with a need to create products that are both beautiful and useful.
